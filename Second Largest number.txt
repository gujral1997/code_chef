Second Largest number

You have beeen provided with 't' number of test cases and 'n' number of integers.
You have to find the second largest number among all the given 'n' number of integers.
if second largest number doesnt exsist, then print 'not exsist' as output.

Constraints
1<=t<=10
1<=n<=10
1<number<10^6

input:
input the number of test cases 't' in the first line.
input the number of integers 'n' you want to enter in second line.
enter all the integers seperated with spaces in the third line.

output:
the second largest number among the given numbers(only one test case output in one line)

sample test case 1:
input:
2
3
1 2 3
4
5 8 1 3
output:
2
5

3
1 1 1
output:
not exsist
